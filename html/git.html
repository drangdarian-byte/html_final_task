Git

  • git clone — создаёт локальную копию удалённого репозитория.
  • git status — показывает состояние файлов в рабочей директории и области подготовленных изменений.
  • git add — добавляет изменения в область подготовленных файлов перед созданием коммита.
  • git commit -m "Сообщение" — создаёт коммит с описанием внесённых изменений.
  • git push — отправляет локальные коммиты в удалённый репозиторий.
